Windows 10 содержит огромное количество дерьмовых функций. Она все больше превращается
из системы Windows NT в рекламно-коммерчески-шпионскую помойку.
Большинство юзеров воспринимают систему windows как черный ящик.
Виндовс делает одно, виндовс делает другое. Но в реальности эта система состоит из компонент.
Каждое действие можно разобрать. Где, какой процесс это делает, как его отключить или блокировать.
Автор попытался найти и вырезать всякое дерьмо не экстенсивными методами, а попытавшись понять источники проблемы
и точки воздействия на них.

Данные отключалки предназначены для тех, кто уже решил для себя следующее :

1) Приватность важнее функциональности. Готов отказаться от некоторых функций и фич ради этого.
2) Не хочу быть частью облачной системы microsoft, моя система локальна на подобии windows 7.
3) Любой стук системы в сеть, если он не понятен, деанонимизирует или тем более сливает приватную инфу, вреден.
4) Не буду никогда пользоваться магазином, metro приложениями, учетной записью microsoft, кортаной, onedrive.
5) Реклама и установка чего-либо без моего согласия недопустимы.

Если ваши взгляды с этим расходятся, не запускайте батники без предварительного изучения - вы можете потерять
некоторые функции. Да и в любом случае лучше просмотрите их, прежде чем запускать.

Рекомендую пользоваться редакцией Windows 10 Enterprise LTSB/LTSC.
В обычной версии microsoft будет постоянно обновлять функционал, клепать новые билды, которые по сути
приближаются к установке поверх windows 11,12,...
В LTSB можно расчитывать на статичность системы, функции обновляться не будут. Только безопасность и багфиксы.

Под администратором запустите :
 win11_eradicate_windefend.cmd - средство перманентного отключения defender для windows 11. необходимо выполнить до запуска всего остального.
 win10_disable_shit_ALL.cmd - отключение основных нежелательных стучалок и функций
 extra\win10_remove_all_metro_apps.cmd - удалить все метро приложения, кроме магаза
 extra\win10_remove_store.cmd - удалить магаз
 extra\win10_ws_licensing_disable.cmd - отключить лицензирование metro приложений и связанный с этим стук.
  Осторожно, могут быть проблемы при создании новых юзеров - не появится "edge", "настройки", система не сможет установить для нового
  юзера никакие modern приложения вообще, даже встроенные !
 extra\win10_disable_wuauserv.cmd - отключить обновления винды на домашней редакции (Windows 10 core)
 extra\win10_disable_online_license_validation.cmd - заткнуть онлайн активацию через firewall
 extra\calculator_classic - установить классический калькулятор
 extra\taskmgr_classic - установить диспетчер задач из windows 7
